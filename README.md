# L'Étoile - Premium Luxury Restaurant Website

A production-quality, responsive luxury fine dining restaurant portfolio website. Built with custom CSS, modular JavaScript, GSAP animations, Lenis smooth scrolling, and Swiper.js.

## Tech Stack
- **HTML5 & CSS3**: Semantic elements, custom variables, responsive media queries, and premium typography.
- **GSAP (GreenSock Animation Platform)**: Staggered list animations, text reveals, and ScrollTrigger reveals.
- **Lenis Smooth Scroll**: High-performance momentum-based scrolling.
- **Swiper.js**: Highly responsive testimonial slider.
- **Vanilla ES6+ JavaScript**: Dynamic render grid, live search and multi-filtering system, favorites system using `localStorage`, and custom keyboard-controlled lightbox modal.

## Folder Structure
- `/assets/images/`: Curated gourmet photography and restaurant interior backgrounds.
- `/css/`: Modular stylesheets (`style.css`, `animations.css`, `responsive.css`).
- `/js/`: Modular interactions (`main.js`, `animations.js`, `menu.js`).
- `/pages/`: Subpages (`menu.html`).
- `index.html`: Baseline homepage.

## Features (Phase 1 & 2)
1. **Transparent & Sticky Navigation**: Beautiful transition from floating layout to frosted glass sticky layout on scroll.
2. **Parallax Hero Section**: Smooth load staggers with GSAP for typography reveals.
3. **Interactive Menu Grid**: Instant search by dish title, ingredients, or category. Multi-selectable filter chips for dietary guidelines (vegetarian, vegan, gluten-free) and attributes (bestseller, seasonal, spicy).
4. **Interactive Favorites System**: Save and persistence of favorites using the heart icon and `localStorage`.
5. **Interactive Lightbox Modal**: View detailed dish descriptions, preparation times, ingredients, and calories with keyboard navigation support (Left/Right arrows, Escape key).
